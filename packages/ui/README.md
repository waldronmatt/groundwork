# Demo-UI

A demo react component library.

## Installation

Install dependencies:

```bash
pnpm add @waldronmatt/demo-ui
```

## Getting Started

```tsx
import { Button, Link } from '@waldronmatt/demo-ui';

function App() {
  return (
    <>
      <Button>Hello</Button>
      <Link href="/">World</Link>
    </>
  );
}

export default App;
```
